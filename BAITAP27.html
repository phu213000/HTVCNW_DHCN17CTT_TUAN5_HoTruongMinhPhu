Cascading Style Sheets(CSS) is a style sheet language used for describing the look and formating of document written ina markup language . Although most often used to change the style of web pages and user interfaces written in HTML and XHTML , the language can be applied to any kind of XML document , including plain XML, SVG and XUL . Along with HTML and JavaScript , CSS is cornerstone technology used by most website to create visually engaging webpages , user interfaces for web applications , and user interface for many mobile applications.
