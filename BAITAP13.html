Border Style

This property must be used if you are using any of the Border properties.

Without using this border-style property you won't see any borders at all.

Syntax : border-style (top)(right)(bottom)(left)

Exmaple: border-style :soild dotted groove double

Here the top border is green has a soild style,

right border is yellow one has dotted,

the bottom has blue one has groove,

and Left border is pink one has the double style
